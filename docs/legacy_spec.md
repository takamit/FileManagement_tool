# 既存機能仕様メモ

## 概要

本ツールは、対象フォルダと比較フォルダを走査し、差分確認、更新候補抽出、削除候補整理、バックアップ付き実行を行うためのファイル管理ツールです。

## 主な機能

### スキャン
- フォルダ再帰走査
- 除外ディレクトリ指定
- 除外拡張子指定
- 同名・差分の判定補助

### 実行
- ゴミ箱移動
- 更新処理
- バックアップ
- 元に戻すための履歴管理

### UI
- GUI による一覧表示
- フィルタ
- 進捗表示
- 実行前確認ダイアログ

## 設計メモ

- UI は `ui/` に配置
- 実処理は `core/logic` と `core/services` を通す
- 例外や進捗は `core/utils` で補助する
