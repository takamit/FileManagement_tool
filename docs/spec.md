# ファイル管理ツール 仕様書

## 概要
本ツールは、2つのフォルダ（対象フォルダ / 比較フォルダ）を比較し、
ファイルの差分検出・削除・更新・整理を安全かつ効率的に行う Python 製 GUI アプリケーションである。

## 機能仕様
### スキャン
- フォルダ再帰スキャン
- UI 非ブロッキング
- フィルタ適用対応
- スキャン完了後の再描画

### 判定
- 同一ファイル
- 更新候補
- 同名同サイズ
- 同名別内容
- サイズ差分

### zip 対応
- zip は展開しない
- 内部ファイル一覧のみ取得する
- zip 内 zip は無視する
- 読み取り失敗はスキップする

### 実行
- 削除（ゴミ箱）
- 更新（move）
- バックアップ対応
- Undo 対応

## 設計原則
- UI は `ui/`
- ロジックは `core/`
- copy 禁止、move 必須
- 表示されている行のみ操作対象
- 例外処理必須
