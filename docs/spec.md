# ファイル管理ツール 仕様書

## 概要
本ツールは 2 つのフォルダ（対象 / 比較）を比較し、差分検出・削除・更新・整理を安全かつ効率的に行う GUI アプリケーションである。

## 機能仕様
### スキャン
- フォルダ再帰スキャン
- UI 非ブロッキング
- フィルタ適用対応
- スキャン完了後に再描画

### 分類
- 同一ファイル
- 更新候補
- 同名同サイズ
- 同名別内容
  - 対象側が大きい
  - 比較側が大きい
  - 同サイズだが内容違い
- サイズ差分

### zip 対応
- zip は展開しない
- 内部一覧のみ取得
- zip 内 zip は無視
- 読み取り失敗はスキップ

### 実行
- 削除（ゴミ箱）
- 更新（move）
- バックアップあり
- Undo 対応

## 設計原則
- copy 禁止、move 必須
- 表示行のみ操作対象
- UI / ロジック分離
- 例外処理必須
