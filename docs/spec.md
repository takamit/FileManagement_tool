# ファイル管理ツール 仕様書

## 概要

本ツールは2つのフォルダを比較し、差分検出・整理・更新を行うGUIアプリケーションである。

---

## 機能仕様

### スキャン

* 再帰スキャン
* UI非ブロッキング
* フィルタ対応

### 分類ロジック

* 同一ファイル
* 更新候補
* 同名同サイズ
* 同名別内容
* サイズ差分

### zip処理

* 展開しない
* 一覧取得のみ
* zip内zip無視

### UI

* フィルタ（表示行のみ対象）
* ソート
* 色分け
* 選択制御

### 実行処理

* 削除（ゴミ箱）
* 更新（move）

#### 更新ルール

* copy禁止
* 必ずmove
* バックアップ必須
* Undo対応

---

## 設計ルール

### フィルタ原則

表示されている行のみ操作対象

### UI / ロジック分離

* GUI：app/gui.py
* ロジック：app/scanner.py / executor.py

### zip原則

* 展開禁止
* 再帰禁止
