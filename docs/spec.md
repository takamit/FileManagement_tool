\# ファイル管理ツール 仕様書（Spec）



\## 1. 概要



本ツールは、2つのフォルダ（対象 / 比較）を比較し、

ファイルの差分検出・整理・更新を安全かつ効率的に行う

Python製GUIアプリケーションである。



\---



\## 2. 目的



\* フォルダ間の差分を明確化

\* 不要ファイル削除の効率化

\* 更新処理の安全な実行

\* 大量ファイル環境での安定動作



\---



\## 3. 機能仕様



\### 3.1 スキャン機能



\* フォルダ再帰スキャン

\* UI非ブロッキング（スレッド処理）

\* フィルタ適用対応

\* スキャン完了後の再描画



\---



\### 3.2 ファイル分類ロジック



以下のカテゴリに分類する：



\* 同一ファイル

\* 更新候補

\* 同名同サイズ

\* 同名別内容



&#x20; \* 対象側が大きい

&#x20; \* 比較側が大きい

&#x20; \* 同サイズだが内容差異

\* サイズ差分



\---



\### 3.3 zip対応（軽量モード）



\* zipは展開しない

\* 内部ファイル一覧のみ取得

\* 画像拡張子のカウント

\* zip内zipは無視（再帰禁止）

\* 読み取り失敗はスキップ



\---



\### 3.4 サマリー表示



表示項目：



\* 対象フォルダ件数

\* 比較フォルダ件数

\* ファイル数比較結果（対象 / 比較 / 同数）

\* 画像数（通常 + zip内）

\* 各分類の件数



\---



\### 3.5 UI仕様



\#### フィルタ



\* 完全対応

\* 表示行のみ操作対象（重要）



\#### 表示



\* 色分け（分類別）

\* ソート（列クリック / ボタン）



\#### 選択操作



\* 全選択（フィルタ反映）

\* サイズ比較選択

\* 選択解除（対象側 / 比較側）



\---



\### 3.6 実行処理



\#### 対応操作



\* 削除（ゴミ箱）

\* 更新（移動）



\#### 更新仕様（厳守）



\* copy禁止

\* 必ずmoveで実行

\* 更新後、元ファイルは削除

\* バックアップ生成（Undo可能）



\---



\### 3.7 バックアップ



\* セッション単位で管理

\* manifestによる追跡

\* Undo対応



\---



\### 3.8 プログレス制御



\* スキャン / 実行の進捗表示

\* 中断時リセット

\* UIフリーズ防止



\---



\### 3.9 設定管理



\* 変更がある場合のみ保存確認

\* 無変更時は即終了



\---



\## 4. 設計ルール（必須）



\### 4.1 フィルタ原則



表示されている行のみ操作対象とする



\---



\### 4.2 更新処理原則



\* copy禁止

\* 必ずmoveを使用



\---



\### 4.3 zip処理原則



\* 展開禁止

\* 一覧参照のみ

\* 再帰禁止



\---



\### 4.4 UI / ロジック分離



| 種別   | ファイル            |

| ---- | --------------- |

| GUI  | app/gui.py      |

| ロジック | app/scanner.py  |

| 実行処理 | app/executor.py |



\---



\## 5. 既知の不具合



\### 5.1 未実装



\* `\_move\_replace\_with\_backup` が未定義



&#x20; \* executor側の実装漏れ



\---



\## 6. 拡張予定



\* 差分ハッシュ判定（精度向上）

\* 非同期I/O導入

\* 大容量対応（数十万ファイル）

\* ドラッグ＆ドロップ対応

\* 画像プレビュー機能



\---



\## 7. 非機能要件



\* 大量ファイルでも安定動作

\* UIフリーズ禁止

\* 例外処理必須

\* ログ出力対応

\* 既存仕様の破壊禁止



\---



\## 8. 開発ルール



\* 未完成コードの出力禁止

\* 動作保証必須

\* 既存仕様を壊さない

\* 表記ゆれ禁止

\* UIとロジック分離

\* 例外処理必須



\---



\## 9. 注意事項



本ツールはファイルの削除・移動を伴うため、

重要データに対して使用する場合は事前バックアップを推奨する。
