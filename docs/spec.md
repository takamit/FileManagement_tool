# ファイル管理ツール 仕様書

## 1. 概要
本ツールは、2つのフォルダ（対象フォルダ / 比較フォルダ）を比較し、ファイルの差分検出・削除・更新・整理を安全かつ効率的に行う Python 製 GUI アプリケーションである。

## 2. 目的
- フォルダ間の差分を明確化する
- 不要ファイル削除を効率化する
- 更新処理を安全に実行する
- 大量ファイル環境でも安定動作させる

## 3. 機能仕様
### 3.1 スキャン機能
- フォルダ再帰スキャン
- UI 非ブロッキング
- フィルタ適用対応
- スキャン完了後の再描画

### 3.2 ファイル分類
- 同一ファイル
- 更新候補
- 同名同サイズ
- 同名別内容
  - 対象側が大きい
  - 比較側が大きい
  - 同サイズだが内容違い
- サイズ差分

### 3.3 zip 対応（軽量モード）
- zip は展開しない
- zip 内部のファイル一覧のみ取得
- 画像拡張子をカウント
- zip 内 zip は無視
- 読み取り失敗はスキップ

### 3.4 実行処理
- 削除（ゴミ箱）
- 更新（move）
- バックアップあり
- Undo 対応

## 4. 設計ルール
- フィルタ時は表示行のみ操作対象
- 更新処理は copy 禁止、move 必須
- zip は展開禁止、一覽参照のみ、再帰禁止
- UI とロジックを分離する
- 例外処理必須

## 5. 既知の課題
- `app/gui.py` が大きく、将来的な分割候補
- `ui/` は旧構造の互換資産として残置中
- 更新処理のロールバックは堅牢化済みだが、実運用で再確認が必要

## 6. 今後の整理方針
- まず現行機能を保持したまま復旧する
- その後に `app/gui.py` の責務分割を行う
- `ui/` は未使用が確認できた段階で削除する
