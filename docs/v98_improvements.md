# v98 改善パッチ

今回の追加実装は、前回の即時修正に続く「完成度を大きく底上げする」ための 3 本柱です。

## 1. GUI 文言の整合
- 既定の更新は **安全コピー** であることを GUI 上で明示
- 旧来の「移動更新」注意書きを差し替え
- 確認ダイアログ・詳細欄でも **安全コピー** と表示

## 2. レポート出力の強化
- 実行レポートを **txt / json / csv / html** の 4 形式で保存
- 変更予定レポートの GUI 出力を追加
- 履歴レポートの GUI / CLI 出力を追加

## 3. 履歴の可視化
- 直近 3 件ではなく、保持されている履歴を一覧表示
- 操作内訳、件数、dry-run を確認可能
- 選択した履歴の詳細表示とレポート出力に対応

## 追加 CLI
- `--action list-history`
- `--action export-history --undo-manifest <manifest.json>`
